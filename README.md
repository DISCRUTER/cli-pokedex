An easy and cody way to explore the world of pokemon.

> This project was made as a part of a course. And hereby, will only receive occasional updates when I have time. If you want a feature added feel free to modify it.  

## Usage
This following are the list of supported commands.
- **`catch`**
    - Catch the specified pokemon.
    - **Usage**: 
        ```bash
        $ cli-pokedex > catch <pokemon-name>
        ```
- **`explore`**
    - List of the pokemons encountered in the area.
    - **Usage**: 
        ```bash
        $ cli-pokedex > explore <area-name>
        ```
- **`inspect`**
    - Inspect all the caught pokemon.
    - **Usage**: 
        ```bash
        $ cli-pokedex > inspect <pokemon-name>
        ```
- **`map`**
    - List 20 next location areas | Go to next 20.
    - **Usage**: 
        ```bash
        $ cli-pokedex > map
        ```
- **`mapb`**
    - List 20 previous location areas | Go to previous 20.
    - **Usage**: 
        ```bash
        $ cli-pokedex > mapb
        ```
- **`pokedex`**
    - List all the pokemon caught.
    - **Usage**: 
        ```bash
        $ cli-pokedex > pokedex
        ```
- **`help`**
    - Display all the commands and their usage.
    - **Usage**: 
        ```bash
        $ cli-pokedex > help
        ```
- **`exit`**
    - Exit the Pokedex.
    - **Usage**: 
        ```bash
        $ cli-pokedex > exit
        ```
